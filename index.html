À propos du projet

Le projet consiste en la mise en place d'un pipeline CI/CD pour déployer un site web statiquesur AWS. J'ai utilisé les technologies suivantes :

  • Git et GitHub : Pour la gestion du code source
  • AWS CodeBuild : Pour la construction et le test
  • AWS S3 : Pour le stockage et la distribution
